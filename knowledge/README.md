# Base de Conhecimento do ChefIA

Este diretório contém o "cérebro técnico" do seu bot. O ChefIA lê todos os arquivos `.md` desta pasta para usar como contexto em suas respostas.

## Como adicionar conteúdo
1. Crie um novo arquivo `.md` (ex: `confeitaria.md`).
2. Adicione informações técnicas, dicas, processos ou receitas baseadas em técnicas.
3. O bot passará a usar essas informações automaticamente na próxima mensagem que receber.

## Dicas para conteúdo de qualidade
- Use uma linguagem técnica mas explicativa.
- Foque no "porquê" (técnica) e não apenas no "o quê" (ingredientes).
- Adicione curiosidades ou dicas de "Chef" para tornar a resposta mais humana.
